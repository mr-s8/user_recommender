# User Recommender for HumHub
A module for HumHub that adds friend recommendations to the homescreen.
Recommendations are generated based on common activity in Humhub Spaces (user groups). In addition commen likes can be taken into account, but this mechnism isnt fully finished yet.


## Features
- adjust time and frequency of recommendation generations
- adjust similarity threshold to only get meaningful recommendations
- highlight recommendations that are reciprocal (two users get each other shown as recommendation)
- set arbitrary text for an info popup
